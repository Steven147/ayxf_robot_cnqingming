# ayxf_robot

`ayxf_robot`是一个基于飞书API的自动化后端服务，旨在提高团队的工作效率，通过自动化的方式处理日常任务和数据交换。

## 功能特性

- 自动回复消息
- 日程管理
- 数据统计和报表生成
- 自定义工作流程自动化
